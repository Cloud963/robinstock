# 1. POR QUE ESTE PROJETO É RELEVANTE?

## 1.1 Importância para a Competição Robinhood

O RobinStock representa a evolução natural da missão original da Robinhood: **democratizar o acesso aos mercados financeiros**. Enquanto a Robinhood revolucionou o trading ao eliminar comissões e simplificar a experiência do usuário, o RobinStock leva essa democratização ao próximo nível através da tecnologia blockchain.

Nosso projeto é relevante porque:

- **Alinhamento com a Visão Robinhood**: Continuamos a missão de tornar investimentos acessíveis a todos, agora sem fronteiras geográficas ou barreiras bancárias tradicionais
- **Inovação Tecnológica**: Utilizamos smart contracts e AMM (Automated Market Makers) para criar um sistema de trading 24/7, sem intermediários
- **Timing Perfeito**: O mercado de RWAs (Real World Assets) está projetado para atingir $16 trilhões até 2030 (Boston Consulting Group)

## 1.2 Qual Problema Estamos Resolvendo?

### Problemas do Sistema Tradicional:

| Problema | Impacto | Nossa Solução |
|----------|---------|---------------|
| Horário limitado de trading | Mercados fecham às 17h, fins de semana sem operação | Trading 24/7/365 |
| Barreiras geográficas | Investidor brasileiro não pode comprar ações americanas facilmente | Acesso global instantâneo |
| Liquidação lenta (T+2) | Dinheiro preso por 2 dias úteis | Liquidação instantânea |
| Altas taxas de corretagem | Custos que corroem retornos | Taxas mínimas (0.3%) |
| Fracionamento limitado | Ações caras inacessíveis | Tokens infinitamente divisíveis |
| Custódia centralizada | Risco de contraparte | Self-custody com carteiras Web3 |

## 1.3 Proposta de Valor

**"Trade Any Stock, Anytime, Anywhere - Powered by Blockchain"**

Nossa proposta de valor se baseia em três pilares:

1. **Acessibilidade Universal**: Qualquer pessoa com uma carteira crypto pode negociar ações tokenizadas
2. **Eficiência Operacional**: Liquidação instantânea, sem intermediários, custos reduzidos
3. **Transparência Total**: Todas as transações são verificáveis on-chain

A integração entre mercados financeiros tradicionais (TradFi) e blockchain (DeFi) cria uma ponte que combina o melhor dos dois mundos:

```

### Como Funciona a Tokenização:

1. **Custódia**: Ações reais são mantidas por um custodiante regulado
2. **Emissão**: Tokens ERC-20 são emitidos 1:1 representando as ações
3. **Oracle**: Preços são atualizados via oráculos descentralizados (Chainlink)
4. **Trading**: Usuários negociam tokens em pools de liquidez AMM
5. **Resgate**: Tokens podem ser queimados para resgatar ações reais

### Para Investidores:

- **Self-Custody**: Suas ações, suas chaves, seu controle
- **Composabilidade DeFi**: Use ações como colateral para empréstimos

### Para o Mercado:
- **Liquidez Global**: Pool de liquidez acessível mundialmente
- **Eficiência de Capital**: Menos capital parado em processos de liquidação
- **Redução de Custos**: Eliminação de intermediários
- **Inovação**: Novos produtos financeiros possíveis

# 5. DEFINIÇÃO DE RWA (Real World Assets)

## 5.1 O Que São RWAs?

**Real World Assets (RWAs)** são ativos do mundo físico ou financeiro tradicional que são representados como tokens em uma blockchain. Essa representação digital permite que ativos tradicionalmente ilíquidos ou de difícil acesso sejam negociados de forma eficiente no ecossistema DeFi.

### Definição Técnica

> RWAs são tokens blockchain que representam propriedade ou direitos sobre ativos tangíveis ou intangíveis existentes fora da blockchain, mantendo uma conexão verificável com o ativo subjacente.

## 5.2 Categorias de RWAs

```

## 5.3 Como Ações se Encaixam na Categoria RWA

Ações são um dos casos de uso mais naturais para tokenização RWA porque:

### Características Ideais

1. **Fungibilidade**: Uma ação TSLA é igual a outra ação TSLA
2. **Divisibilidade**: Já são naturalmente divisíveis
3. **Preço Verificável**: Oráculos podem buscar preços em tempo real
4. **Regulamentação Existente**: Framework legal já estabelecido
5. **Demanda Global**: Investidores do mundo todo querem acesso

### Implementação no RobinStock

```solidity
// Exemplo simplificado de token de ação
contract TSLAToken is ERC20, Ownable {
    // 1 TSLA token = 1 ação Tesla
    // Backed 1:1 por ações em custódia
    
    address public oracle;  // Chainlink price feed
    address public custodian;  // Custodiante regulado
    
    function mint(address to, uint256 amount) external onlyOwner {
        // Só pode mintar quando ações são depositadas
        _mint(to, amount);
    }
    
    function burn(uint256 amount) external {
        // Queima tokens para resgatar ações reais
        _burn(msg.sender, amount);
    }
}
```
